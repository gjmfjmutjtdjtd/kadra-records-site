Файл: readme.txt

Проект: KADRA Records — лендинг студии звукозаписи
Описание: Лендинг для продвижения услуг записи, сведения, мастеринга, продажи битов и продюсирования.
Версия: 1.0
Дата: 2025-08-11

Список файлов:
- index.html         — основной HTML-файл
- style.css          — стили
- script.js          — JS (включает AJAX для формы)
- handler.php        — PHP-обработчик формы (пишет в leads.csv)
- leads.csv          — файл с заявками (создаётся автоматически при первой заявке)
- robots.txt         — правила для роботов
- sitemap.xml        — карта сайта
- favicon.svg        — иконка
- readme.txt         — этот файл
- /images/           — папка с изображениями (поместите сюда ваши фото и видео)
- /audio/            — папка с примерами треков

Как разместить (пошагово)
1) Подготовка:
   - Переименуйте домен в файлах sitemap.xml и readme.txt, замените https://your-domain.ru на ваш реальный домен.
   - Поместите реальные изображения в папку images/ (hero-bali.jpg, studio-moscow.jpg, studio-volgograd.jpg, hero-loop.mp4)
   - Поместите аудио-предпрослушивания в /audio/ (sample1.mp3, sample2.mp3)

2) Загрузка на хостинг (рекомендуется cPanel или FTP):
   - Зайдите в панель хостинга -> File Manager -> public_html (или root вашего сайта)
   - Загрузите все файлы и распакуйте, если отправляете ZIP
   - Убедитесь, что handler.php и leads.csv доступны для записи:
     - Рекомендуемые права на leads.csv: 664 или 666 (в редких случаях)
     - handler.php не должен быть исполняемым битом; стандартные права 644

3) Проверьте работу формы:
   - Откройте сайт и заполните форму
   - Если всё верно, файл leads.csv появится в корне и будет содержать новую строку
   - Если нет — проверьте логи сервера и права доступа

4) Настройка почты (опционально):
   - Для уведомлений по email раскомментируйте блок mail() в handler.php
   - Убедитесь, что хостинг поддерживает функцию mail() или настройте SMTP (PHPMailer)

5) Настройка SSL:
   - Рекомендуется подключить Let's Encrypt через панель хостинга

6) CDN и производительность:
   - Для продакшена: оптимизируйте изображения (WebP), используйте gzip/ Brotli, подключите кеширование

7) Частые проблемы:
   - Форма не отправляет: проверьте права файла leads.csv и доступ к handler.php
   - leads.csv пустой: проверьте блокировку flock и права на запись
   - Видео не проигрывается: убедитесь, что hero-loop.mp4 загружен и поддерживаем кодек

Как скачать заявки (leads.csv)
- Через File Manager в панели хостинга скачайте /public_html/leads.csv
- Или подключитесь по FTP/SFTP и скачайте файл
- Откройте в Excel или Google Sheets (разделитель — запятая)
